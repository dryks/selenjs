ОБЩИЕ ЗАМЕЧАНИЯ
==============================================
Скрипт предназначен для проведения онлайн тестирования по правилам дорожного движения РФ (категории AB).
Условия использования скрипта можно найти на сайте http://pdd-test.hint.su

УСТАНОВКА
==============================================
1. Скопировать директорию pdd-test в любую директорию вашего сайта.

2. На страницу где вы планируете установить данный скрипт добавить следующий код:

<!-- PDD-TEST START -->
<script src="//cdnjs.cloudflare.com/ajax/libs/require.js/2.1.16/require.js" data-main="/путь/к/скрипту/pdd-test/pdd-test.min.js" id="pdd-test-script"></script>
<div id="pdd-test">Загрузка</div>
<!-- PDD-TEST END -->

3. Проверить работоспособность скрипта.


ПРИМЕЧАНИЯ
==============================================
Если ваш сайт использует кодировку CP-1251 либо какую то другую кодировку отличную от UTF-8,
необходимо перекодировать файл с шаблонами templates.html. Это можно сделать с помощью стандартного текстового редактора.
notepad.exe в Windows или geany в Linux.
-----------------------
В файле pdd-test.min.js есть несколько задокументированных настроек, которые вы можете изменить.
-----------------------
Если вы собираетесь вносить какие либо изменения в скрипт, не забудьте сделать себе резервную копию исходных файлов.
-----------------------
Для того чтобы вывести блок со статистикой текущего пользователя, разместите следующий код на странице на которой подключен скрипт.
<div id="pdd-statistics"></div>
